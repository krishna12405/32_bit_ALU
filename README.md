# 32-Bit ALU
## Overview
This project implements a 32-bit Arithmetic Logic Unit (ALU) using Verilog HDL. The ALU performs a variety of arithmetic and logical operations based on a control signal and is designed using a modular RTL approach. The design was verified through simulation using a comprehensive testbench.
## Features
* 32-bit data width
* Arithmetic operations: Addition, Subtraction
* Logical operations: AND, OR, XOR, NOT
* Shift operations: Logical Left Shift and Logical Right Shift
* Comparison operation (Equal or Not)
* Zero flag generation
* Overflow Detection
* Functional verification using a Verilog testbench
## Tools Used
* Verilog HDL
* Xilinx Vivado
* Simulation Testbench
